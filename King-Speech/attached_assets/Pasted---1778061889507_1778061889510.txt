## ЗАДАЧА: Автопереход между уровнями + кнопка выхода (крестик)

### Контекст проекта
Приложение King Speech (Expo + React Native, Expo Router). Сейчас после завершения задания виджет "Уровень пройден!" предлагает вернуться на карту. Нужно изменить навигацию и редизайнить этот виджет.

### Изменения в логике навигации

**Файл: `app/level/[id].tsx`** (и аналогичные экраны заданий)

1. Найти кнопку "На карту" / "Back to map" в виджете завершения уровня.
2. Заменить её на кнопку **"Далее →"** которая переходит на следующий уровень.

Логика перехода к следующему уровню:
```typescript
import { getLevelsData } from '@/constants/gameContent';
import { useLang } from '@/context/LangContext';

// Получить все уровни
const levels = getLevelsData(lang);
const currentIndex = levels.findIndex(l => l.id === currentLevelId);
const nextLevel = levels[currentIndex + 1];

if (nextLevel) {
  router.replace(`/level/${nextLevel.id}`);
} else {
  // Это последний уровень — перейти на карту
  router.replace('/(tabs)/');
}
```

3. Добавить **крестик (×) в правом верхнем углу** виджета завершения уровня:
```typescript
 router.replace('/(tabs)/')}
  style={{ position: 'absolute', top: 16, right: 16, padding: 8 }}
>
  

```

### Редизайн виджета "Уровень пройден!"

Создать модальный виджет (Modal или Animated overlay) с таким содержимым:

**Верхняя часть:**
- Крестик × в правом верхнем углу → возврат на карту `/(tabs)/`
- Большой эмодзи-трофей или SVG-звезда по центру (анимация ZoomIn)
- Текст "Уровень пройден!" / "Level complete!" — крупный, золотой (#FFD166)

**Средняя часть:**
- Карточка с итогами: набрано XP, лучший балл, время выполнения
- Строка со звёздами (1–3 звезды в зависимости от балла: < 5 = 1 звезда, 5–7 = 2 звезды, ≥ 8 = 3 звезды)

**Нижняя часть:**
- Кнопка "Далее →" (полная ширина, золотой фон) → переход к следующему уровню
- Мелкая ссылка "На карту" под кнопкой → возврат в `/(tabs)/`

### Крестики везде — добавить во все игровые экраны

Добавить кнопку выхода (Ionicons "close") в правый верхний угол на следующих экранах. Нажатие → `router.replace('/(tabs)/')`:

- `app/(tabs)/interview.tsx` — добавить крестик в шапку над аватаром Jenny
- `app/showtime-stage.tsx` — добавить крестик в верхний правый угол
- `app/showtime-playback.tsx` — добавить крестик
- `app/level/[id].tsx` — добавить крестик в шапке экрана задания

При нажатии крестика во время активной записи — показать Alert:
```typescript
Alert.alert(
  'Выйти из задания?',
  'Прогресс этого задания не сохранится',
  [
    { text: 'Остаться', style: 'cancel' },
    { text: 'Выйти', onPress: () => router.replace('/(tabs)/') }
  ]
);
```

### Edge cases
- Если текущий уровень последний в базе → кнопка "Далее" ведёт на `/(tabs)/` с сообщением "Все уровни пройдены!"
- Крестик показывать всегда, даже во время загрузки уровня
